# GlucoPilot

GlucoPilot is a personal diabetes data analytics platform for glucose, insulin, meals and exercise tracking.

The project starts from LibreView CSV exports and aims to provide clear visualizations, statistics and pattern analysis for personal diabetes management.

## Goals

- Import glucose data from LibreView CSV files
- Visualize daily glucose curves
- Calculate time in range and other glucose statistics
- Track insulin doses, meals, exercise and daily context
- Detect personal glucose patterns
- Explore predictive models in the future

## Important note

GlucoPilot is intended for personal data analysis and visualization. It does not provide medical advice or insulin dosing recommendations.
